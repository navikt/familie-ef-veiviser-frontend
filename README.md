# familie-ef-veiviser

Veiviser-applikasjon for enslig forsørger.

## Kom i gang med utvikling

### Lokal utvikling
1. `npm install`
2. `npm start`

### Test av prodversjon med Vite server
1. `npm run build`
2. `npm run preview`

### Test av prodversjon med prod-lik server
1. `npm run build`
2. `node ./dist-server/server.js`

## Henvendelser
